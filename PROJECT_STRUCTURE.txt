🚀 快速启动
------------
1. pnpm install
2. cd apps/backend && cp .env.example .env
3. cd ../.. && pnpm prisma:generate
4. pnpm prisma:migrate && pnpm prisma:seed
5. pnpm dev

访问：http://localhost:3000/api-docs
